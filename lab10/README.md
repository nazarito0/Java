Хід виконання роботи

1. Створити Базу даних в СКБД: SQLite, MySQL, PostgreSQL (Oracle XE, MS SQL - 2 останні варіанти за бажанням студента). (mariadb-server)

2. Написати Java-програму, за допомогою якої реалізувати базові CRUD-операції.

3. Продемонструвати результат виконання програми та відобразити результати в звіті з лабораторної роботи.

- бібліотека: автори(ід, ім'я), книги(ід, назва, рік, кількість), жанри (ід, назва),
